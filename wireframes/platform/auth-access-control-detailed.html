Platform Auth & Access Control

인증은 로그인 화면 하나로 끝나지 않는다. 세션 컨텍스트, 재인증, 권한 거부, tenant 전환 안전성을 같은 설계 축으로 다룬다.

Session-aware Step-up required Context-safe

핵심 확인 포인트

  • 권한 부족과 세션 만료가 다른 상태로 보이는지
  • 민감 액션 전 재인증 흐름이 자연스러운지
  • tenant 전환 시 컨텍스트가 섞이지 않는지
