# WI-DS-004 직원 모바일 레이아웃 시스템

## 목적

Tenant Employee 모바일 경험 전반에 공통으로 쓰일 레이아웃 규칙을 정의한다.

## 레이아웃 구조

- 상단 상태 바
- 화면 헤더
- 상태 히어로 카드
- 빠른 액션 그리드
- 작업 / 알림 리스트
- 하단 탭바

## 원칙

- 첫 화면은 상태 확인과 즉시 행동이 동시에 가능해야 한다.
- 엄지 조작 범위 안에 주요 액션을 둔다.
- 스크롤이 길어져도 상단 핵심 상태는 먼저 읽혀야 한다.
- 데스크톱 구조를 단순 축소하지 않는다.

## 공통 패턴

- 상태 히어로 카드
- 2열 빠른 액션 그리드
- 오늘 할 일 리스트
- 이번 주 요약 카드
- 하단 탭바

## 화면 적용 대상

- Employee Home
- New Request Hub
- Signature Inbox
- My Schedule
- My Profile
