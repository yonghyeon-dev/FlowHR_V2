# WI-DM-004 급여 정산 기준과 마감 플로우 상세화

## 작업 ID

- `WI-DM-004`

## 목적

급여 정산과 마감은 급여 화면만의 문제가 아니라, 근태 예외, 휴가 차감, 승인 상태,
문서 완료 여부까지 연결되는 운영 규칙이다.

## 정산 입력 축

- 확정 근태 기록
- 초과근무 승인 결과
- 휴가 차감 결과
- 수당 / 공제 규칙
- 계약 / 지급 기준

## 마감 플로우

### 1. 사전 점검

- 미확정 근태 기록 확인
- 승인 대기 초과근무 확인
- 휴가 차감 충돌 확인
- 급여 기준 변경 문서 확인

### 2. 정산 확정

- 개인별 계산 결과 생성
- 조직별 합계 검토
- 예외 항목 수동 보정

### 3. 마감 완료

- 급여 마감 상태 변경
- 명세서 발행 준비
- 감사 로그 기록

## 위험 항목

- 미확정 근태
- 누락된 승인 문서
- 휴가 차감 불일치
- 수당 규칙 미반영

## 상태 정의

- Draft
- Review Needed
- Ready to Close
- Closed
- Reopened

## 사용자 노출 방식

### Tenant Admin

- Payroll Dashboard
- Closing Watch
- 예외 검토 리스트

### Tenant Employee

- 완료 후 명세서 알림
- 관련 문서 인박스

## 연결 작업

- `WI-TA-003` Attendance 모듈
- `WI-TA-004` Leave 모듈
- `WI-TA-006` Documents 모듈
- `WI-TA-010` Payroll Dashboard / Closing
