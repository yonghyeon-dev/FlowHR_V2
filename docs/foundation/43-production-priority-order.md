# FlowHR_V2 프로덕션 우선순위

## 고정 우선순위

1. SSOT 정렬
2. 권한 SSOT
3. UI SSOT
4. Auth / Prisma / Postgres 실연결
5. 코어 도메인 E2E
6. 모듈 확장
7. QA / 배포 / 출시

## 적용 규칙

- 사용자가 특정 기능만 언급해도 상위 우선순위가 미완료면 먼저 그 항목을 처리한다.
- 부분 구현은 완료로 보지 않는다.
- 와이어프레임용 동작은 프로덕션 기능으로 인정하지 않는다.
- 권한, 화면, 데이터 기준이 연결되기 전에는 완료 처리하지 않는다.

## 현재 즉시 적용 중인 우선순위

1. `WI-1218` 화면별 제어 항목 분류
2. `WI-1219` 와이어프레임 전용 임시 상호작용 제거
3. `WI-1220` 권한 SSOT 완성
