# WI-DM-002 휴가 정책과 차감 규칙 상세화

## 작업 ID

- `WI-DM-002`

## 목적

휴가 정책은 화면 표시에 필요한 참고값이 아니라, 요청 생성, 관리자 승인,
잔여 계산, 만료 안내, 촉진 대상 분류를 움직이는 핵심 도메인 규칙이다.

## 정책 구성 요소

### A. 휴가 유형

- 연차
- 반차
- 시간 단위 휴가
- 보상휴가
- 경조사
- 병가
- 무급휴가

### B. 발생 규칙

- 입사 기준 자동 부여
- 근속 기준 추가 부여
- 수동 지급
- 이월 규칙

### C. 차감 규칙

- 일 단위 차감
- 반일 차감
- 시간 단위 차감
- 근무제/소정근로시간 기준 환산

### D. 만료 및 촉진

- 만료 예정일
- 촉진 대상 조건
- 촉진 알림 시점
- 사용 독려 배치 대상

## 계산 원칙

- 회사 정책과 근무제를 기준으로 계산한다.
- 반차와 시간 단위는 동일한 차감 축으로 환산 가능해야 한다.
- 부여, 사용, 예정, 만료 예정 수치를 분리해서 보여준다.

## 승인 연결 규칙

- 팀 일정 충돌 시 추가 승인 가능
- 증빙 필요 유형은 첨부 강제
- 특정 유형은 관리자만 신청 가능

## 사용자 노출 방식

### Tenant Admin

- 정책 설정 화면
- 휴가 대시보드
- 촉진 대상 리스트

### Tenant Employee

- 요청 허브 정책 힌트
- 잔여 현황 카드
- 신청 폼 내 경고/가이드

## 연결 작업

- `WI-TA-004` Leave 모듈
- `WI-TE-003` 요청 허브
