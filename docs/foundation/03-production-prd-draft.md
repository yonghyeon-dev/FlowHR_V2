# FlowHR_V2 프로덕션 PRD 초안 v1

## 제품 개요

FlowHR_V2는 중소형부터 멀티 법인까지 확장 가능한 HR SaaS를 목표로 한다.
관리자는 근태, 휴가, 문서, 승인, 급여, 리포트를 한 제품 안에서 운영하고,
직원은 출퇴근과 요청, 서명, 확인 업무를 빠르게 처리해야 한다.

## 대상 사용자

- Platform Operator
- Tenant Admin
- Tenant Manager
- Tenant Employee

## 핵심 문제

- HR 운영자는 시스템을 보는 시간이 길고, 반복 처리 업무가 많다.
- 직원은 자주 쓰는 기능은 적지만, 필요한 순간에는 즉시 끝나야 한다.
- SaaS 운영 조직은 고객사 운영과 내부 운영을 같은 콘솔에 섞으면 안 된다.

## 제품 목표

- 운영 병목을 홈에서 바로 드러낸다.
- 부서/조직 단위 예외를 빠르게 처리하게 한다.
- 직원 셀프서비스 완료 시간을 줄인다.
- 권한, 감사, 과금, 지원 운영까지 프로덕션 기준으로 설계한다.

## 범위

### 포함

- Platform 운영 콘솔
- 인사정보, 근태, 휴가, 승인, 문서, 급여, 리포트
- 성과, 채용, 온보딩/오프보딩
- 역할/권한, 감사 로그, 알림, 외부 연동

### 제외

- 이번 단계에서의 실제 백엔드 구현
- 회계 ERP 세부 엔진 구현
- 국가별 법률 엔진 확정

## 성공 기준

- 관리자 홈에서 오늘 처리할 우선순위가 즉시 읽힌다.
- 직원 홈에서 주요 셀프서비스 액션이 1~2탭 안에 끝난다.
- Platform Console이 고객사 운영 화면과 명확히 분리된다.
- WI 단위 산출물이 화면/권한/도메인 기준으로 추적 가능하다.
