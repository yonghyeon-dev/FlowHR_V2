# FlowHR_V2 프로덕션 로드맵 요약

## 목적

이 문서는 프로덕션 완료까지의 큰 흐름을 요약한다.  
세부 기준은 아래 문서를 따른다.

- [45-production-ssot-master.md](./45-production-ssot-master.md)
- [46-production-work-breakdown.md](./46-production-work-breakdown.md)
- [47-production-execution-order.md](./47-production-execution-order.md)

## 단계 요약

1. SSOT 정렬
2. 권한/인증/데이터 SSOT 완성
3. 코어 도메인 E2E 완성
4. 화면별 실제 기능 치환
5. 확장 모듈 완성
6. 품질/배포/출시

## 현재 위치

현재는 `1. SSOT 정렬` 단계다.

즉 지금 우선순위는 기능 추가가 아니라 아래 항목이다.

- 화면별 제어 항목 분류
- 와이어프레임 전용 임시 동작 제거
- 권한 SSOT 완성
