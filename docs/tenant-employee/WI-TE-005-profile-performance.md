# WI-TE-005 내 프로필 / 내 성과 상세 설계

## 작업 ID

- `WI-TE-005`

## 목적

직원은 자신의 인사 정보와 성장 상태를 서로 다른 메뉴에서 흩어 보지 않아야 한다.
`내 프로필`과 `내 성과`는 조회 중심이되, 필요한 행동으로 자연스럽게 이어지는 화면이어야 한다.

## 대상 화면

- `TE-501` My Profile
- `TE-502` My Leave Balance
- `TE-503` My Performance

## 핵심 UX

- 기본 인사정보, 소속, 고용 상태가 한 화면에서 읽혀야 한다.
- 잔여 휴가와 예정 사용 현황은 별도 메뉴 없이 프로필 흐름 안에서 연결돼야 한다.
- 목표, 평가 일정, 1:1 준비 상태는 부담 없이 읽히는 방식이어야 한다.

## 주요 섹션

- 기본 프로필
- 고용 / 조직 정보
- 잔여 휴가 및 예정 사용
- 목표 진행 상태
- 평가 일정
- 최근 피드백 / 1:1 메모

## 주요 액션

- 개인정보 확인
- 증빙 정보 갱신 요청
- 휴가 정책 보기
- 평가 상세 보기
- 1:1 준비 메모 확인
