# WI-TE-001 Employee Home / Mobile Home 상세 설계

## 작업 ID

- `WI-TE-001`

## 목적

직원 홈은 메뉴 진입점이 아니라 `오늘 해야 할 행동 허브`여야 한다.
상태를 먼저 보여주고, 그 다음 필요한 액션을 바로 제안해야 한다.

## 대상 화면

- `TE-001` Employee Home
- `TE-002` Employee Mobile Home

## 핵심 UX

- 출근 상태가 첫 카드여야 한다.
- 휴가 요청, 정정 요청, 문서 서명은 홈에서 바로 시작할 수 있어야 한다.
- 모바일은 데스크톱 축소판이 아니라 한 손 조작 기준으로 다시 설계한다.

## 데스크톱 구성

- Today Status
- My Tasks
- This Week
- My Summary

## 모바일 구성

- 상태 히어로 카드
- 빠른 액션
- 오늘 할 일
- 이번 주 요약
- 하단 탭바
